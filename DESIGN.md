# SASIGLOBAL Design System: "VIP Elite"

## 💎 Identidad de Marca
SASIGLOBAL SAS proyecta una imagen de autoridad, precisión industrial y sofisticación tecnológica. Nuestra estética se aleja de lo genérico para abrazar un diseño "Enterprise High-End".

## 🎨 Paleta de Colores
- **SASI Crimson**: `#dc143c` (Acción, Energía, Precisión)
- **Deep Black**: `#0a0a0f` (Elegancia, Autoridad, Profundidad)
- **Silk Grey**: `#f8f9fa` (Pureza, Estética limpia, Profesionalismo)
- **Pure White**: `#ffffff` (Contraste, Claridad)

## ✍️ Tipografía
- **Display**: `Outfit` o `Space Grotesk`. Tracking de al menos `0.2em`. Peso `900` para títulos de impacto.
- **Body**: `Inter`. Interlineado relajado (`leading-loose`).

## 🏢 Arquitectura de Componentes
- **Orquestación**: Todos los componentes globales (Header/Footer) se inyectan dinámicamente.
- **Glassmorphism**: Uso de `backdrop-blur` y opacidades sutiles para profundidad.
- **Animaciones**: Curvas `cubic-bezier(0.16, 1, 0.3, 1)` para movimientos fluidos y naturales.

## 🚫 Qué NO hacer
- No usar colores genéricos (Azul puro, Rojo puro).
- No usar bordes redondeados (Radius 0 por defecto para estética industrial).
- No saturar de información; el espacio en blanco es lujo.
