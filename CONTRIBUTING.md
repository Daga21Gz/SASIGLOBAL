# Contributing to SASIGLOBAL Platform

We welcome contributions from authorized engineers and strategic partners. To maintain the "VIP Elite" quality standards of this platform, please adhere to the following guidelines.

## 💎 Excellence Standards

1.  **Visual Perfection**: All UI changes must adhere to the 2026 Crystal VIP design system. Use high-fidelity glassmorphism and subtle animations.
2.  **Performance First**: No external dependencies unless strictly necessary. Minimize JavaScript footprint.
3.  **Clean Code**: Maintain semantic HTML5 and organized CSS variables.
4.  **Asset Integrity**: Do not use external placeholders. Generate or provide high-resolution local assets.

## 🛠 Workflow

1.  **Branching**: Use descriptive branch names (`feature/new-module`, `fix/ui-polish`).
2.  **Commits**: Follow conventional commits (`feat:`, `fix:`, `docs:`, `style:`).
3.  **Testing**: Verify responsiveness across all breakpoints (Desktop, Tablet, Mobile).

## 🔒 Security & Confidentiality

All engineering documents and source code within this repository are confidential. Unauthorized distribution is prohibited.
